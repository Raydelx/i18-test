# 1. Install Dependencies

```bash
npm install

```

## 2. Start development Server

```bash
npm run dev

```

### Preview & Build

```bash
npm run preview
npm run build

```

## Project Structure

Inside of your project, you'll see the following folders and files:

```js
/
├── public/
│   └── ...
├── src/
│   ├── components/
│   │   └── ...
│   ├── layouts/
│   │   └── ...
│   └── pages/
│       └── ...
└── package.json
```
